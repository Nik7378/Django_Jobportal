# JobBoard - Job Application Portal

Designed and developed Job  Application Portal using Python, Django Framework, Bootstrap and MySQL as part of a Web Application. Created functionalities for Admin roles. 

## Functionalities

Admin can:
* Add New Jobs
* Search Jobs (based on year of passing, degree, location)
* Update Job list
* Delete Job
* View Job List


## Getting Started

* Python 3+
* MySQL
* Django
* VS Code

## Prerequisites

* Downlaod and create virtual environment for project folder
* Crate MYSQL database with the name given in settings.py file inside app folder
* Create table inside database with the columns as model fields in models.py file
* Perofrm migration to migrate model fields


## Run

* Activate virtual environment and move to app direction sing cmd
* Project runs with python manage.py runserver.
* Click On http://127.0.0.1:8000/
